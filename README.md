# The-Beward-intercom-monitoring-template
Данный шаблон написан для мониторинга домофонов Beward. Я постарался создать универсальный шаблон, чтобы мониторить данные о модеди устройства, серийника+тригер если новый( отслеживать как часто меняли домофон ), мониторить регистрацию sip, работу  магнита и различных данных, такие как объем озу, частоту процессара и т.д.
